# Image Uploads

Upload quiz images under this folder using the paths listed in `src/data/image-shot-list.json`.

Example:

```text
public/images/bleach/zangetsu.jpg
```

When an image is ready, move or copy that shot-list entry into `src/data/image-guess.json` and set:

```json
"image_path": "/AniMayhem/images/bleach/zangetsu.jpg"
```

Use official promotional/key art or project-owned screenshots/crops. Do not hotlink images from other sites.
